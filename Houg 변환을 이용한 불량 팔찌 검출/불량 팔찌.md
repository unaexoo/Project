# 유리 팔찌 데이터를 이용한 불량 검출
- https://www.kaggle.com/datasets/almique/glass-bangle-defect-detection-classification
-> kaggle 데이터 활용
- C#으로 구현
- Windows Froms 앱(.NET Framework)

## 전처리
  - 오츠 이진화 사용 후 가우시안 블러링 적용

## 불량여부
- 허프변환 C#의 라이브러리 활용
- 원이 제대로 검출되지 않는다면 불량, 아니라면 정상으로 판별
  
## 분석
- ACC : 89.47%
- Percision : 88.99%
- Sensitivity : 94.81%
- F1 Score : 91.81%
